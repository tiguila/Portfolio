UAV Package Delivery Simulation System

Simulate a Unmanned Aerial Vehicle (UAV) package delivery system using MATLAB.

Functionalities

  • Take off from point A
  • Land on point B
  • Find optimal path from point A to point B using A* algorithm
  • Detect static and dynamic objects in real time using LiDAR (Light detection and ranging) sensor
  • Deviate from dynamic and static objects in real time using a waypoint follower block
